Managing the Badge Provider

A Badge Provider is a third-party service that provides an API for creating badge classes and badge assertions.

At the moment, we only support Badgr as a badge provider.

Define the properties of your Badgr account here, including your Badgr API key and the URL of your Badgr instance. You can then create badge classes and badge assertions for a particular course from Composer.
